[Visão geral] Trata-se de um projeto desenvolvido por discentes da Universidade de São Paulo, com o objetivo de desenvolver as capacidades de programação, utilizando a Linguagem C. O programa se baseia na implementação do jogo de cartas Rummikub, representado em hexadecimais. Para mais detalhes, consultar o relatório que acompanha os arquivos do projeto.
